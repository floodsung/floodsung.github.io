强化学习集成

多模态能力

联合文本-视觉训练，支持跨模态推理

技术创新

核心突破

  • 实现了与 OpenAI o1 正式版相当的数学、代码和多模态推理能力
  • 在短推理链模式下显著超越 GPT-4o 与 Claude 3.5 Sonnet
  • 简化 RL 框架，无需复杂的蒙特卡洛树搜索

研究发现

在 RL 训练过程中，Flood Sung 团队观察到模型能力提升的同时，生成 token 的数量也随之增加——这是模型自发涌现的现象 （来源）。 这一洞见与 DeepSeek 等同行机构的发现相契合。
